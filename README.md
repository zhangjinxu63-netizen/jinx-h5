# 公众号 H5 版

这是适合放进公众号菜单、文章「阅读原文」或自动回复链接的移动端 H5 页面。

## 本地预览

直接双击 `index.html` 可以看静态效果；如果浏览器拦截 BTC 实时接口，建议用本地服务预览：

```powershell
cd wechat-h5
python -m http.server 8080
```

然后打开：

```text
http://127.0.0.1:8080
```

## 放进公众号

上线后需要一个 HTTPS 网址，例如：

```text
https://你的域名/index.html
```

然后在公众号后台配置：

- 自定义菜单：菜单内容选择「跳转网页」
- 图文文章：放到「阅读原文」
- 自动回复：回复 H5 链接

## 正式上线前建议

当前页面前端已经预留了展示结构，但完整实时指数估值最好通过你自己的 HTTPS API 返回。原因是公众号 H5 在微信内置浏览器里运行，外部接口可能会遇到跨域、稳定性或访问限制。
